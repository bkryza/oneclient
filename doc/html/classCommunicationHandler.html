Detailed Description

The CommunicationHandler class.

Object of this class represents dynamic (auto-connect) TCP/IP connection to cluster. The CommunicationHandler allows to communicate with cluster by sending and receiving ClusterMsg messages. TCP connection is fully automatic, which means that event if current CommunicationHandler isnt connected, it will connect automatically. If it is, it will reuse previously opened socket.
